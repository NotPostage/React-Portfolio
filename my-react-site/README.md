# My React Site

A small portfolio-style React site scaffolded with Vite. This project demonstrates a minimal React + Vite setup with a simple navigation, a Home page, and a Projects page. Styles are plain CSS and components live under `src/components` and `src/pages`.

## Features

- Built with React and Vite for fast development and HMR
- Simple component structure: `Nav`, `Home`, `Projects`
- Scoped CSS files for components and pages (no CSS framework)

## Project structure (important files)

- `index.html` — app HTML entry
- `src/main.jsx` — app bootstrap
- `src/App.jsx` — main app component and route container
- `src/components/Nav.jsx` (+ `Nav.css`) — navigation bar
- `src/pages/Home.jsx` (+ `Home.css`) — home page
- `src/pages/Projects.jsx` (+ `Page.css`) — projects page
- `src/*.css` — global and page styles

## Get started

- Install dependencies:

```bash
npm install
```

- Start dev server with HMR:

```bash
npm run dev
```

- Build for production:

```bash
npm run build
```

- Preview the production build locally:

```bash
npm run preview
```

These scripts follow the standard Vite template. If you're using yarn or pnpm, replace `npm` with your package manager.
